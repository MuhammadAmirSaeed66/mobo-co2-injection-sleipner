# Sleipner plotting data

The optional plotting script expects parsed plume-boundary CSV files in:

```text
data/sleipner_2019/CSV_exports/
```

Each CSV is expected to contain:

- `SEG_ID`
- `X_EASTING`
- `Y_NORTHING`
- `Layer`

and to use a filename ending in `_parsed.csv`.

Only upload these source data when their redistribution terms permit it.
Otherwise, provide the official download source, access date, preprocessing
instructions, and checksums here.

Official source used by the manuscript: Equinor / CO2DataShare, Sleipner 2019
benchmark dataset. Add the exact download URL and access date before release.
